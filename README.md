# SPI devices handle

## Main Purpose
	To provide interfaces with various spi devices on Beaglebone capes as well as GPIO capabilities.

## Current Version
### Ver: 0.1

## Currently Implemented

* BME280 atmospheric chip
 - Barometric sensor
 - Humidity sensor
 - Temperature sensor

* BMX055 positional chip
 - Accelerometer module
